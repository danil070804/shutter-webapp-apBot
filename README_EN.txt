SHUTTER ISLAND BOT

QUICK START:
1. Run INSTALL_PACKAGES.bat (first time only)
2. Run START_BOT.bat

OR just run START_BOT.bat - it will install and run.

CONFIGURATION:
Edit config.py before running:
- TOKEN = "your_bot_token" (from @BotFather)
- ADMIN_IDS = (123456789,) (your ID from @userinfobot)
- ADMIN_CHAT_ID = -100... (group ID for applications)

TROUBLESHOOTING:
If "Python not found" - install Python 3.10+ from python.org
Make sure to check "Add Python to PATH" during installation.

FILES:
- bot.py - main bot code
- config.py - settings (EDIT THIS)
- notifications_pro.py - notifications module
- requirements.txt - package list
- START_BOT.bat - run this to start bot
- INSTALL_PACKAGES.bat - install packages only
